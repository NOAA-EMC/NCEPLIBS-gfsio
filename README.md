# GFSIO

 API to convert GFS Gaussian output into grib output


### Prerequisites

Compilers: GNU | Intel | Clang | AppleClang 


### Installing

```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/path/to/install /path/to/NCEPLIBS-gfsio
make -j2
make install
```
